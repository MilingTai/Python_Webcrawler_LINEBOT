前置作業
-
**0.To install django:** <br>
```pip install django```

**1.To Create Project:**
```
cd.. your_file_location
django-admin startproject YourProjectName

cd project_location
py manage.py runserver
```
**2.To Create App:**
```
py manage.py startapp appName
```
